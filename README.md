# Fluid Ounce and Milliliter Converter

A simple Python unit converter that converts measurements between fluid ounces and milliliters. The program allows the user to choose the desired conversion, enter the amount, and instantly displays the converted value. It uses separate user-defined functions for each conversion, making the code easy to understand and maintain while demonstrating the use of functions, conditional statements, and basic arithmetic operations.

## Features

* Converts **fluid ounces (fl oz)** to **milliliters (mL)**.
* Converts **milliliters (mL)** to **fluid ounces (fl oz)**.
* Uses separate functions for each conversion.
* Accepts user input for conversion type and amount.
* Displays the converted result in a simple format.

## How to Run

1. Make sure Python 3 is installed.
2. Run the script:

   ```bash
   python converter.py
   ```
3. Select the conversion type by entering:

   * `1` for Fluid Ounce → Milliliters
   * `2` for Milliliters → Fluid Ounce
4. Enter the amount to convert.
